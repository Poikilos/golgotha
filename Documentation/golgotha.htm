Golgotha Real Time 3D engine

 

This is a (very temporary) page that describes how to download and install the Golgotha Engine source code together with the required data files. If you are in trouble, feel free to send an email to Patrick Grawehr concerning your problems. 

The game engine Golgotha was originally developed by crack.com (spoken as "crack dot com") around 1998. The team of crack.com also developed the much better known game "Abuse", which is part of most linux distributions around nowadays. Unfortunately, the Golgotha was never finished because the team was out of money and crack.com went bankruptcy. The members of the team made an interesting decision at this time and released all their code and data they already had to the public. 

I've now been working on this code for quite some time. Several improvements were added and most bugs fixed. 

System requirements for Golgotha

The core of the Golgotha engine is quite old, so you don’t need to have one of those latest high-end gaming towers to get it running.

Hardware and Operating System

To execute Golgotha, you need:

§         Windows 95/98/2000/XP (With software rendering, you’ll get it probably to run even under NT4) or

§         Linux (with at least a 2.x kernel, I suppose) or,

§         Solaris (in development) or, if you have the source code release,

§         Any other platform that is implemented or, if you know how to code,

§         Any imaginable platform (including game stations like the PS2).

§         At least 32MB RAM (128 MB recommended)

§         A fast CPU (well, I can’t give a speed here, since it depends on the platform). For Intel, 400MHZ should do a quite good job.

§         A graphics card, preferably hardware accelerated, with DirectX or OpenGL support and the required drivers installed. Some amount of video memory would also be a good start.

§         For Windows or linux, at least 100MB of free Hard disk space is recommended although Golgotha can be configured to run completely from cd.

§         (development only) You need about 700MB free Hard disk space to install and compile the complete source with all extras.

§         A mouse and a keyboard (If your platform supports these, use anything that matches on a gaming platform like PS2)

§         (optional) A stereo sound card would be tasty. Attach it to your stereo amplifier to get the most out of the great soundtrack of this game.

§         (optional) a CD-ROM drive

§         (optional) a Network card for LAN games or a modem for Internet games.

If you do not know which platform (operating system) applies to you, it’s most probably windows.

Special Software requirements

Windows:

§         DirectX 5 or newer is required. You get the latest version from http://www.microsoft.com/directx or most discs that accomply your preferred gaming magazine.

§         If you didn’t download a full installation package, be sure to have the Visual Studio runtime files installed (“msvcrt.dll”). These are automatically installed if you install Visual Studio or some Office program like MS Word.

§         (development only) Microsoft Visual C++ 6.0 SP4 or newer. Visual Studio .NET should work, too. 

§         (development only) The Microsoft DirectX SDK (can be downloaded from Microsoft's web-site) At least Version 7. If you have a newer version of the DX SDK (at this time, DX9 is out), be sure to also have the V7 header files and libraries.

Linux:

§         An OpenGL package for hardware support. Otherwise, mesa can do software rendering, although this is slower. 

§         An X server must be running.

§         (development only) gcc must be installed and working. The mesa-devel package and the X11 development header files are required.

§         (development only) Some plain text editor will be great. So make yourself familiar with emacs or vi or whatever you like.

 

Download and installation instructions:

The following files make out the current Golgotha distribution (Version 1.0.5.6). The first time you install Golgotha, you have to download all of these, later on, we will sporadically update some of them (like the code alone.)

To install Golgotha, create a directory on your HD (like d:\golgotha) and unpack everything into this folder. Be sure to rebuild the directory structure stored in the zip file. When you are done, you should have a lot of subfolders in the main directory. Now open up golgotha.dsw with MSVC and compile everything. On Linux, type "gmake" at the prompt. More instructions are included in the Manual.

G021224.zip The Source code itself.

textures.zip All files that make up the textures for the engine.

bitmaps.zip The Bitmaps folder. Contains elements for the user interface. 

objects.zip The Objects. Contains the model files for the 3d units.

resource.zip The resources. Contains configuration data.

levels.zip The Game Level data. 

Manual.zip Some important information about the engine. This is far from complete, but already gives an overview of the source code. Please read this first.
